# Oppgaver til forelesning 1 i ITF31619 webapplikasjoner

## Oppgave 1: User stories
Lagde [5 user stories](oppgave1_user_stories.md) til en eksamensapplikasjon.

## Oppgave 2: Flytdiagram
Lagde et [flytdiagram](oppgave2_flowchart.png) for flyten til en deltaker i eksamensapplikasjonen.

## Oppgave 3: Wireframe
Lagde 4 wireframes for administratorsidene i eksamensapplikasjonen. En wireframe for [en landingsside hvor man kan logge inn eller registrere seg](oppgave3_wireframes/wireframe_sign_in_or_sign_up.png), en for [en side hvor man kan opprette eksamner](oppgave3_wireframes/wireframe_create_exam.png), en for [en side hvor man kan se eksamenene man har laget i et grid-format](oppgave3_wireframes/wireframe_exam_grid_view.png), og en for [en side hvor man kan redigere tittel og beskrivelse til en eksamen og se resultatene til deltakerene som har gjennomført eksamene](oppgave3_wireframes/wireframe_edit_exam_and_user_stats.png).

## Oppgave 4: DB-diagram
Lagde [et DB-diagram for eksamensapplikasjonen](oppgave4_db-diagram.png).

## Oppgave 5: Trello
Lagde [en tavle i Trello med tre boards, og la til tre user stories fra oppgave 1](oppgave5_trello/trello_tavle.png). La til en sjekkliste med ett punkt, og en forfallsdato til hver user story ([user story 1](oppgave5_trello/trello_sjekkliste_og_forfallsdato_1.png), [user story 2](oppgave5_trello/trello_sjekkliste_og_forfallsdato_2.png), [user story 3](oppgave5_trello/trello_sjekkliste_og_forfallsdato_3.png)). La også US'ene til [en epic](oppgave5_trello/trello_epic.png).

## Oppgave 6: GitHub
Opprettet dette GitHub repositoriet, og lastet opp alle filene knyttet til denne oppgaven her.

## Oppgave 7: Installere programvare
Lastet ned og installerte Visual Studio Code, Node.js og Git. [Noterte ned versjonsnumrene til programmene her](oppgave7_installere_programvare/versjonsnumre.md), og tok [et screenshot av versjonsnumrene her](oppgave7_installere_programvare/versions_screenshot.png).
